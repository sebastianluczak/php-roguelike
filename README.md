## Run & build

#### Prerequisites:
- Docker with Docker Compose
- Terminal Emulator capable of displaying rich text and XTERM256 support

```shell
cd ~
git clone https://github.com/sebastianluczak/php-roguelike.git
cd php-roguelike
chmod +x bin/*
bin/build.sh
```
